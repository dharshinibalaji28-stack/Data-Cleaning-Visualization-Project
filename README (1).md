# Data Cleaning & Visualization Project

## Objective
Clean a raw dataset and visualize insights using Python.

## Dataset
Titanic Dataset

## Steps Performed
1. Removed duplicate records.
2. Handled missing values in Age and Embarked columns.
3. Performed exploratory data analysis.
4. Created visualizations:
   - Survival Count
   - Gender vs Survival
   - Passenger Class vs Survival

## Key Findings
- More passengers did not survive than survived.
- Female passengers had a higher survival rate than males.
- First-class passengers had better survival rates compared to lower classes.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
